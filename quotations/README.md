# Quotations Folder

Store vendor quote PDFs here using timestamp-from-source naming.

## Naming Rule

`YYYY-MM-DDTHH-MM_vendor_quote.pdf`

For the Omega / Mega Services quote discussed, preferred filename:

`2025-11-04T12-54_omega-services_quote.pdf`

The source timestamp comes from quote text:

`Tue, Nov 4, 2025 at 12:54 PM`

If timezone is unknown in source material, keep timezone unknown in related JSON metadata.
